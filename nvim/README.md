# Neovim configuration

 The configuration uses [lazynvim](https://www.lazyvim.org/installation) in order to install plugins
 
 plugins used:
 - telescope
 - snacks
 - lualine
 - catppuccin (theme
    

  You can edit the file located at ``~/.config/nvim/init.lua`` in order to make changes to neovim.

  In order to edit the file you can use:
```bash
  nvim ~/.config/nvim/init.lua
  ```
